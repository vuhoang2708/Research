# Thư mục Nghiên cứu (Research Workspace)

Thư mục này được sử dụng làm không gian làm việc để nghiên cứu các `idea` (ý tưởng) công nghệ mới và thử nghiệm các `repository` (kho chứa mã nguồn) mã nguồn mở.
